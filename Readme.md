这是一个基于STM32F103C8T6的健康监测系统，用来检测心率和运动数据。



\# 功能

1. 手电筒：按键控制led
2. 时钟：RTC时钟+断电计时
3. 测步：MPU6050采集数据➡STM32处理➡OLED显示数据
4. 心率血氧：MAX30102采集数据➡STM32处理➡OLED显示数据
5. WiFi：



\# 模块

* STM32F103C8T6：主控芯片

&#x20;

* MAX30102：检测心率血氧

&#x20;

* &#x20;MPU6050：检测步数

&#x20;

* &#x20;OLED：显示检测数据



* ESP8266：WiFi数据传输，串口通信

&#x20;
